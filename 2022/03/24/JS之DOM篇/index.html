元素

获取元素

标签,属性,文本,注释
getElementById()根据id获取元素
getElementByTagName()根据标签名获取元素
getElementByClassName()根据类名获取元素
querySelector()制定选择器
获取body元素
document.body
获取html元素
document.documentElement

操作元素

innertText标签里面的文字内容,去除空格和换行
innerHTML包括html标签,保留空格和换行
src
href
id , alt , title

表单元素的属性操作

type, value, checked, selected, disabled
className修改css样式
this.className = “change”

排他思想

自定义属性

element.属性
element.getAttribute(“属性名”)
自定义属性index=”1”
element.getAttribute(“属性名”)
设置属性值
element.属性=”值”
div.id=”test”
element.setAttribute(“属性”,”值”)

H5自定义属性

data-index=”1 “

节点操作

节点包括元素节点,文本节点(空格,换行),属性节点,注释节点,所有html元素(节点)均可以被创建,修改,删除

父子级

node.parentNode

子节点

parentNoe.children 返回所有子元素节点
parentNode.firstElementChild 第一个元素节点
parentNode.firstElementChild 最后一个元素节点
parentNode.childNodes 所有的子节点,包含元素节点,文本节点
parentNode.firstChild 第一个子节点包含文本节点
parentNode.lastChild 最后一个子节点包含文本节点

兄弟节点

node.previousElementSibling上一个元素兄弟
node.nextElementSibling下一个元素兄弟
node.previousSibling上一个兄弟,包含文本节点
node.nextSibling下一个兄弟,包含文本节点

创建节点

node.creatElement(child) node为父级,child为子级

添加节点

node.appendChild(child)添加到父节点末尾
node.insertBefore(child)添加到父节点前面

删除节点

node.removeChild(child)
阻止链接跳转javascript:;

复制节点

node.cloneNode()

DOM重点核心

文档对象模型(Document Object Model)
通过dom获取过来的是一个对象,所以称为文档对象模型
主要有创建,增,删,改,查,属性操作,事件操作

创建

1.document.weite
2.innerHTML
3.createElement

删

1.removeChild

改

1.修改元素属性:src,href,title
2.修改普通元素内容:innerHTML,innerText
3.修改表单元素:value,type,disable
4.修改元素样式:style,className

查

1.dom提供的API方法:getElementById,getElementByTagName古老用法不推荐
2.h5新增方法：querySelector,querrySelectAll
3.利用节点操作元素: fu(parentNode),子(children),兄(previousElementSibling,nextElementSibling)

属性操作

1.setAttribute:设置dom的属性值
2.getAttribute:得到dom的属性值
3.removeAttribute移除属性

鼠标事件

onclick鼠标点击左键触发
onmouseover鼠标经过触发
onmouseout鼠标离开触发
onfocus鼠标获得焦点触发
onblur失去鼠标焦点触发
onmousemove鼠标移动触发
onmouseup鼠标弹起触发
onmousedown鼠标按下触发

事件高级

事件三要素

事件源,事件类型,事件处理程序

事件注册的两种方式：

1.传统方式


2.事件侦听注册事件

删除事件

1.传统方式
eventTarget.onclick = null


2.方法监听注册方式
eventTarget.removeEvventListener(type,listerner[,useCapture])

dom事件流

往下传播：捕获阶段doument,html ,body,div
往上传播:冒泡阶段div,body,html,document

事件对象event

事件的一系列相关数据的集合,鼠标,键盘
clientX鼠标距离浏览器窗口X的坐标
clientY鼠标距离浏览器窗口Y的坐标
target 绑定事件的元素
type 绑定的事件类型
pageX鼠标距离文档x坐标
pageY鼠标距离文档y坐标
screenX鼠标距离电脑屏幕x坐标
screenY鼠标距离电脑屏幕y坐标

e.target和this的区别

  1. e.target 返回的是触发事件的对象（元素） this 返回的是绑定事件的对象（元素）

比如说ul和li如果绑定的是ul当点击li的时候,e.target返回的是li,而this返回的是ul

阻止默认行为

e.preventDefault()链接不跳转

阻止事件冒泡

e.stopPropagation()阻止事件冒泡

事件委托

事件委托的核心原理：给父节点添加侦听器， 利用事件冒泡影响每一个子节点

禁止选中文字和右键菜单

常见的键盘事件

onkeyup 按键被松开时候触发
onkeydown 按键被按下时候松开
onkeypress 按键被按下时候触发 不能识别功能键
