入口函数

DOM 对象 和 jQuery对象

DOM 对象： 用原生js获取过来的对象就是DOM对象
jQuery对象： 用jquery方式获取过来的对象是jQuery对象。 本质：通过$把DOM元素进行了包装
相互转换:
DOM对象转换为 jQuery对象
$(“div”)
jQuery对象转换为DOM对象
1.$(“div”)[index]
2.$(“div”).get(index)

常用API

jQuery选择器

$(“#id”)id选择器
$(“*”)全选选择器
$(“.class”)类选择器
$(“div”)标签选择器
$(“div,p,li”)并集选择器
$(“li,current”)交集选择器

隐式迭代

隐式迭代就是把匹配的所有元素内部进行遍历循环，给每一个元素添加css这个方法

jQuery筛选选择器

:first 第一个元素
:last 最后一个元素
:eq(index) 索引号获取元素
:odd 索引号为奇数的元素
:even 索引号为偶数的元素

筛选方法

parent()查找父级
children(selector)查找最近一级亲儿子
find(selector) 查找所有
siblings(selector)查找兄弟
nexAll() 查找当前之上所有元素
prevtAll() 查找当前之后所有元素
eq.(index) 索引号index的元素

链式编程

$(this).css(“color”, “red”).siblings().css(“color”, “”);

jQuery样式操作

操作css方式

1.参数只写属性名，则返回属性值
$(this).css(“width)
2.参数是属性名，属性值则是设置一祖样式
$(this).css(“color”,”red”)

操作类

addClass(“类”)添加类 不需要小点,要引号
removeClass()删除类
toggleClass()操作类

jQuery效果

显示隐藏

show([speed],[easing],[fn])速度，切换效果，回调函数 显示
hide([speed],[easing],[fn]) 隐藏
toggle([speed],[easing],[fn]) 切换

滑动效果

slideDown([speed],[easing],[fn]) 下滑
slideUp([speed],[easing],[fn]) 上滑
slideToggle([speed],[easing],[fn]) 切换滑动
动画队列
stop()停止动画 写在动画前面

淡入淡出效果

淡入 fadeIn()
淡出 fadeOut()
淡入淡出切换 fadeToggle()
修改透明度 fadeTo() 这个速度和透明度要必须写

自定义动画

animate(params,[speed],[easing],[fn])
params 样式属性 width 以对象形式

属性操作

获取设置元素固有属性
prop(“属性名)
prop(“属性”,”属性值”)

设置获取自定义属性

index=1
attr(“属性”) //类似原生getAttribute
attr(“属性”,”属性值”) //类似原生setAttribute()

内容文本

html()获取元素内容
text()获取文本内容
val()获取表单值

each方法

$(“div”).each(function(index,domle){}) domle是原生 要用$(domle)转换为jQuery
1.each()遍历元素好点
$(“div”).each(function(index,domle){})
2.$.each遍历数据,处理数据好点
$.each($(“div”),function(i,ele){})

创建添加删除元素

创建

var li = $(“

  • 我是后来创建的li
    • “);

    添加元素

    append()后面添加
    prepend()前面添加
    before()外部添加

    删除元素

    html(“”) 删除所有子节点
    remove()删除匹配的元素
    empty()删除匹配元素里面的子节点

    jQuery尺寸方法

    1. width() / height() 获取设置元素 width和height大小 不含padding

    2. innerWidth() / innerHeight() 获取设置元素 width和height + padding 大小

    3. outerWidth() / outerHeight() 获取设置元素 width和height + padding + border 大小

    4. outerWidth(true) / outerHeight(true) 获取设置 width和height + padding + border + margin

      jQuery位置方法

    5. offset()获取设置距离文档的位置（偏移）
    6. position()获取距离带有定位父级位置（偏移） 如果没有带有定位的父级，则以文档为准

    scrollTop()被卷去的头部
    scrollLeft()被卷去的左侧

    事件

    事件注册

    $(“div”).click(function(){事件处理程序})
    常见事件 mouseover,mouseout,blur,focus,change,keydown,keyup,resize,scroll
    事件绑定on()可以绑定多个事件
    事件解绑off()
    自动播放事件trigger()
    拷贝对象$.extend([deep])
