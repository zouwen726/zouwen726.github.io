表单域属性
action 处理表单的服务器url地址 url地址
method 设置表单提交方式get/post
name 定义表单域名称
input输入表单
type值有：
text 文本内容
password 定义密码
radio 单选按钮 ，两个按钮要有相同的name才可以多选一
checkbox 复选框
button用户点击按钮
reset 重置按钮
submit 提交按钮
maxlengh 输入的最大字段
checked=checked 默认选中
label标签
为input标签标注，label标签绑定表单元素，光标自动转到表单上面
select下拉表单元素
