CSS常用属性

发表于2022-03-19|更新于2023-04-28
|阅读量:

选择器

1.标签选择器 p span
2.类选择器 class 类选择器多类名要用空格隔开
3.id选择器id
4.通配符选择器*

字体属性

font-familly 字体样式
font-size 字体大小
font-weight 字体粗细
font-style 字体样式 italic倾斜

文本属性

color文本颜色
text-align文本对齐
text-decoration文本装饰
text-indent文本缩进
line-height行间距

Emment语法

text-align center 可以输入 tac
text-decoration 可以输入 tdn

复合选择器

1.后代选择器
ul li 元素1是父级，元素2是子级
2.子选择器
元素1>元素2 父级下的亲儿子，孙子不选
3.并级选择器
元素1，元素2 中间加格逗号
4.链接伪类选择器
a：link选择所有未被访问的链接
a：visited 选择所有已被访问的链接
a: hover 选择鼠标位于其上的链接
a：active 选择鼠标按下未弹起的链接
5.：focus伪类选择器
当input获得光标时选择

元素显示模式

  • 块元素
    h1-h6,p,div,ul,ol,li
  • 行内元素
    a,strong,b,em,i,del,s,ins,u,span
  • 行内块元素
    img,input,td

显示模式转换

转换为块元素 display：block
转换为行内元素 display：inline
转换为行内块 display：inline-block

css背景

背景颜色background-color：颜色值
背景图片background-image：url（） no-repeat
背景图片尺寸background-size(100%,100%)
背景图片固定
background-attachment
scroll背景图像随内容滚动
fixed背景图像固定
背景颜色半透明
background： rgba（0，0，0，0.3）

css3三大特性

层叠性，继承性，优先级

层叠性：就近原则
继承性：子标签继承父标签文字相关样式
优先级：！important无穷大
权重大小
行内样式1000
id选择器100
类，伪类选择器10
元素 标签 1

盒子模型

1.边框
边框粗细 border-width
边框样式border-style 实线solid虚线dashed点线dotted
简写border： 1px solid pink；
合并相邻边框border-collapse： collapse
2.内边距padding
padding： 5px 10PX 15X 上5 左右10 下20
3.外边距margin
盒子居中对齐
margin： 0 auto；
4.外边距塌陷：父和子同时设置margin会塌陷
解决方案
1.为父元素定义上边框
2.为父元素定义内边距
3.为父元素添加overflow：hidden
清除内外边距
padding 0 margin 0
5.圆角边框 border-radius
正方形50%是个圆
6.盒子阴影
h-shadow水平距离
v-shadow垂直距离
blur模糊距离
spread应用尺寸
color颜色
box-shadow： h-shadow v-shadow blur spread color
文字阴影和盒子差不多

css浮动

无论行内或行内块浮动后具有行内块元素特性
清除浮动
overflow：hidden，auto或 scroll
定位 模式加边偏移 子绝父相
相对定位position：relative
相对原来的位置，不脱标，原来位置继续拥有
绝对定位position：absolute
不占有原来位置，脱标
固定定位
position：fixed
固定版心右侧：左走50%再走版心一半
粘性定位
position：sticky
绝对定位，固定定位会压住下面标准流浮动不会
元素显示与隐藏
display：none不占有原来位置
visibility：hidden 占有原来的位置
overflow： hidden对盒子溢出的内容隐藏

精灵图

字体图标

三角制作

border：50px solid transparent
border-left-colorP：green

鼠标样式

cursor：default默认
default默认
poointer小手
move移动
text文本
not-allowed禁止

其它

表单默认蓝色边框outline：none
防止文本域拖拽resize：none
行内块与文字垂直居中vertical-align
顶线（top ）
中线（middle）
基线（base）
底线（bottom）
图片底侧有空白缝隙，原因是行内块元素和文字基线对齐
解决方案
1.添加vertical-align：middle top bottom
2.转换为块级元素display：block

css3的2D转换之位移

transform: translate(xy)
transform: translateX(n)
transform: translateY(y)

css3的2D转换之旋转

transform:rotate(360deg)
css3的2D转换之中心点
中心点transform-origin:left bottom
css3的2D转换之缩放
transform:scale(0.5)表示上下左右缩放0.5

动画

定义动画，调用动画
定义@keyframe 动画名称{0%{}100%{}}
调用动画animation：name
时间anmation-duration：持续时间

动画常用属性

animation-dely:动画什么时候开始，默认是0
animation-iteration-count播放次数默认是1无限循环infinite
animation-direction是否进行逆向播放alternte
3D转换
3D移动transform:translate3d(x,y,z)
透视perspective:500px 透视写在父盒子身上，旋转木马改变不同图片大小
3D旋转transform:rotate3d(x,y,z)
开启3D空间 transform-style:preserve-3d写给父亲
