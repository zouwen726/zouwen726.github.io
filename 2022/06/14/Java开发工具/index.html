IDEA

idea项目结构

项目 模块 包 类

内容辅助键

  • psvm 快速生成main方法
  • sout 快速输出语句
  • ctrl+ alt+space 代码补全
  • ctrl+/ 单行注释
  • ctrl+alt+L 格式化

Maven

项目的根目录

1.src 源码

​ —main 主工程代码

​ —java主工程的代码

​ —resource 主工程的配置文件

​ —webapp web项目的资源（jsp/html/web-inf）

​ —test 测试代码

​ —java 测试代码

​ —resource 测试需要使用的配置文件

2.pom.xml 项目核心配置文件

maven常用命令

compile

mvn compile 会在项目目录下产生target文件，这是java编译的字节码文件

clean

mvn clean 清理文件 把编译的target文件删除

package

mvn package 将web文件打包成war包

install

mvn install 将当前项目安装到本地仓库中去

maven生命周期

清理 编译 测试 打包 安装 站点发布

坐标

pom.xml

通过pom.xml的坐标去本地仓库找对应的jar包

groupld 组织名称

atifactld 模块名称

version 版本号
