java安装

JRE：java程序的运行环境,包含JVM(保证程序跨平台)和运行时所需要的核心库

JDK：是java程序开发工具包，包含JRE和开发人员使用的工具

开发工具：编译工具（javac.exe）和运行工具（java.exe）

JDK的目录

bin该路径存放jdk的各种命令,javac和java

conf该目录存放jdk相关配置文件

include该目录存放平台的一些特定的头文件

jmods该路径存放jdk各种模块

legal该路径存放jdk各模块的授权文件

lib该路径存放jdk工具的一些补充jar包

java基础

关键字

  • 关键字字母全部小写
  • 常用代码编辑器对关键字有颜色标记 50个左右

常量

在程序运行过程中值不可以发生改变

  • 字符串常量:用双引号括起来的内容
  • 整数常量：不带小数的数字
  • 小数常量：带小数的数字
  • 字符常量：单引号括起来的内容
  • 布尔常量：true false
  • 空常量：一个特殊的值，空值null

数据类型

基本数据类型：数值型和非数值型

  • 数值型 整数（byte，short，int，long）浮点数（float，double）字符（char）
  • 非数值型 布尔（boolean）

引用数据类型

类（class） 接口（interface） 数组（[]）

常用方法

array.add(object)添加元素

array.add(2,”java”)指定位置添加元素

array.remove(object)移除指定元素

array.remove(int index)根据索引移除元素

array.size()返回集合中元素个数

array.set(int index,e element) 根据索引修改元素

继承

zi extends fu

子类所有构造方法默认访问父类无参构造方法

子类继承父类要重写父类所有方法

原因：

子类继承父类数据，子类初始化之前要对父类数据进行初始化

每一个子类构造方法第一句默认super（）代表调用父类构造方法没有带参数说明调用父类无参构造方法

类不能继承多个类

权限修饰符

private 只能本类访问

void 默认 不同包下不能访问，同一包中子类无关类

protect 不同包的子类

public 不同包的无关类

修饰符

final 被final修饰的变量不能改变 修饰的方法不能重写 final 修饰的基本数据类似值不能变 fianl 修饰的引用数据类型值可以改变

因为引用数据类型是地址，地址不能变的里面的值可以改变

static 为所有的对象访问的成员定义为静态成员，静态成员可以用对象名访问也可以用类名访问

访问特点：非静态成员方法可以访问静态成员和非静态成员

​ 静态成员只能访问静态成员变量和成员方法

多态

有继承和实现关系

有方法重写

有父类引用指向子类对象

多态访问成员变量访问的是父类的成员变量

抽象类

没有方法体的方法为抽象方法，类中如果有抽象方法要定义为抽象类 抽象类可以有普通方法 抽象类不能直接创建对象 抽象类有构造方法

接口

抽象的，不能直接创建对象

普通类实现接口要重写接口方法

抽象类实现接口不用重写接口方法
