Hexo 安装

hexo
hexo
发布日期:   2022-01-12
文章字数:   741
阅读时长:   2 分

安装Node.js

安装

Node.js官方地址 https://nodejs.org/en/

验证

安装完成后，要查看我们是否安装成功，可以打开命令提示符（Win + R）

输入 cmd 打开控制台，输入如下命令，如果出现对应版本号，说明安装成功了；

node -v
npm -v

安装Git

安装

官方网址 https://git-scm.com/download/

验证

安装完成后在命令提示符中输入git —version验证是否安装成功。

git —version

注册Github账号

注册

官网地址 https://github.com/

创建个人仓库

  • 点击GitHub中的New repository创建新仓库
  • 仓库名：zouwen726.github.io(必须 用户名.github.io)看黑板这是重点

git 与 GitHub 账号绑定

  • 右键打开 Git Bash，然后输入下面命令：

git config —global user.name “zouwen726”
git config —global user.email “2535685121@qq.com”

上面user.name和user.mail替换自己的即可

配置密钥

作用：用于连接GitHub上传博客

首先在本地生成 ssh 密钥文件，输入如下命令后直接三次回车即可

    ssh-keygen -t rsa -C "2535685121@qq.com"

  • 一般执行上述命令之后，会生成 id_rsa 和 id_rsa.pub两个文件，前者是我们私有的，而后者则是对外开放的

  • 接着找到生成的.ssh 的文件夹中的 id_rsa.pub 密钥，将内容复制；

  • 有的人可能pub文件夹打不开可以用以下方法:

    1.在c盘找到.ssh文件，然后进去输入cmd


    2.在里面输入type id_rsa.pub 然后就能看到密钥，复制密钥 

type id_rsa.pub

  • 打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key:
  • 将刚复制的内容粘贴到key那里，title随便填，保存

  • 输入ssh -T git@github.com，如果如下图所示，出现你的用户名，那就成功了。 
 ssh -T git@github.com

如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，
看到这个信息说明SSH已配置成功！

安装Hexo

  • 在合适的地方新建一个文件夹，用来存放自己的博客文件，比如我的博客文件都存放在C:\blog目录下
  • 定位到该目录下，输入npm i hexo-cli -g安装 
npm i hexo-cli -g
  • Hexo安装完后输入hexo -v验证是否安装成功 
hexo -v

初始化博客

初始化博客

hexo init

然后生成静态博客文件 

hexo g

本地预览博客

hexo s

在浏览器中输入 http://localhost:4000

如果出现以下画面就说明成功了！

修改博客配置

  • 本地博客测试成功后，就是上传到 GitHub 进行部署，使其能够在网络上访问

  • 安装 hexo-deployer-git 

hexo-deployer-git

  • 打开博客根目录下的_config.yml文件，这是博客的配置文件，在这里你可以修改与博客相关的各种信息。

  • 修改最后一行的配置： 

  deploy:
  type: git
  repository: https://zouwen726.github.io
  branch: main

最后

推送到github 

hexo d
